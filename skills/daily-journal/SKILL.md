---
description: Создаёт дневниковую запись с адаптивными вопросами на основе недавних заметок vault
---

Пользователь хочет сделать дневниковую запись в Obsidian vault.

Запусти команду:
```bash
notes-manager daily-journal $ARGUMENTS
```

Флаги:
- `--questions N` — количество вопросов (по умолчанию 5)
- `--find-connections` — искать связи с прошлыми дневниковыми записями
- `--no-create` — только показать вопросы, не создавать файл

Команда сгенерирует адаптивные вопросы на основе недавних заметок и соберёт ответы интерактивно. После выполнения файл сохранится в папку `10-daily/` vault.
