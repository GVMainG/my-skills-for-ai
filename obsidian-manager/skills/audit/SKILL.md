---
description: Проверяет Obsidian vault на проблемы: битые ссылки, дубли, неверные теги, проблемы с frontmatter
---

Пользователь хочет сделать audit заметок в Obsidian vault.

Запусти команду:
```bash
notes-manager audit $ARGUMENTS
```

Если путь к папке не указан в аргументах, используй путь из конфига (запусти без аргументов — команда возьмёт vault из `~/.notes-manager/config.yaml`).

Флаги:
- `--all` — все проверки (по умолчанию)
- `--check-format` — только frontmatter и длина
- `--check-links` — только битые [[ссылки]]
- `--find-duplicates` — только дубликаты
- `--check-tags` — только теги не из иерархии
- `--auto-fix` — автоисправление тривиальных проблем

После выполнения кратко резюмируй найденные проблемы и предложи следующие шаги.
