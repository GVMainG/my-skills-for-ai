---
description: Ищет заметки в Obsidian vault по семантическому запросу через теги, ключевые слова и бэклинки
---

Пользователь хочет найти заметки в Obsidian vault.

Запусти команду:
```bash
notes-manager search "$ARGUMENTS"
```

Если запрос не указан, попроси пользователя уточнить, что искать.

Флаги:
- `--max-results N` — количество результатов (по умолчанию 10)
- `--include-backlinks` — расширить поиск через бэклинки
- `--scope <папка>` — искать только в указанной папке
- `--tags <тег1,тег2>` — фильтр по тегам

После выполнения прокомментируй результаты: почему эти заметки релевантны, есть ли между ними связи.
