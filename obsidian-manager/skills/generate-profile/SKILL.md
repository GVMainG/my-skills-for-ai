Пользователь хочет создать или обновить профиль на основе заметок Obsidian vault.

Запусти команду:
```bash
notes-manager generate-profile $ARGUMENTS
```

Флаги:
- `--update` — обновить существующий CLAUDE.md
- `--output <путь>` — кастомный путь сохранения (по умолчанию CLAUDE.md в корне vault)
- `--sections <список>` — конкретные секции (skills, portrait, graph, interests, context)

Команда делает deep semantic analysis всех заметок: технические навыки, психологический портрет (Big Five, когнитивный стиль, ценности, противоречия), граф знаний, кластеры тем. Результат — файл CLAUDE.md.

После выполнения кратко опиши ключевые инсайты из сгенерированного профиля.
