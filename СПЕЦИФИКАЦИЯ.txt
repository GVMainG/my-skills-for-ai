## Executive Summary

**Название:** notes-manager  
**Версия:** 1.0.0 (MVP)

## Команда 1: Analyze Tags

### Синтаксис

```bash
notes-manager analyze-tags <filepath>
  --auto-apply      # Автоматически применить рекомендации
  --dry-run         # Показать изменения без применения
```

### Функциональность

**Input:**
- Путь к заметке (.md файл)

**Process:**
1. Парсинг файла: извлечь frontmatter, content, текущие теги
2. Загрузка валидных тегов из `TAGS_HIERARCHICAL.md`
3. Semantic analysis содержимого:
   - Keyword extraction (ключевые слова/фразы)
   - Mapping keywords → теги из иерархии
   - Определение основного тега (главная тема заметки)
4. Сравнение текущих тегов с рекомендованными:
   - Теги на удаление (не соответствуют содержимому)
   - Теги на добавление (отсутствуют, но релевантны)
   - Новые теги (если контент не вписывается в существующую иерархию)
5. Валидация по правилам (1-5 тегов, двухуровневая структура)

**Output (текстовый формат):**
```
📝 Анализ тегов: /path/to/note.md

Текущие теги (3):
  #it/csharp
  #личное/дневник
  #образование/конспекты

❌ Удалить:
  #личное/дневник
  Причина: Содержимое не содержит личных рефлексий, фокус на технических концептах

✅ Добавить:
  #it/многопоточность
  Причина: 15 упоминаний "async/await", "Task", "threading"
  
  #it/архитектура

  #it/performance
  Обоснование: Контент фокусируется на оптимизации производительности
  Рекомендация: Добавить в TAGS_HIERARCHICAL.md под #it/performance

```

**Если `--auto-apply`:**
- Автоматически обновить frontmatter
- Сохранить файл
- Вывести "✅ Теги обновлены"

**Алгоритм mapping keywords → tags:**
```python
def map_keywords_to_tags(keywords, tags_hierarchy):
    """
    keywords: ['async', 'await', 'threading', 'C#', 'performance']
    tags_hierarchy: parsed TAGS_HIERARCHICAL.md
    
    Returns: [(tag, confidence_score), ...]
    """
    candidates = []
    
    for tag in tags_hierarchy:
        tag_keywords = extract_tag_keywords(tag)  # Из описания тега
        score = calculate_overlap(keywords, tag_keywords)
        if score > 0.3:  # Порог релевантности
            candidates.append((tag, score))
    
    # Сортировка по score, топ-5
    return sorted(candidates, key=lambda x: x[1], reverse=True)[:5]
```

```python
def check_format(note):
    issues = []
    
    # Проверка frontmatter
    if not note.has_frontmatter():
        issues.append({
            'type': 'missing_frontmatter',
            'severity': 'high',
            'auto_fix': True,
            'fix': 'add_default_frontmatter'
        })
    
    # Проверка длины
    if len(note.content) < config.audit.min_note_length:
        issues.append({
            'type': 'too_short',
            'severity': 'medium',
            'auto_fix': False,
            'suggestion': 'Expand content or move to inbox'
        })
    
    # Проверка названия
    if note.filename in config.audit.default_untitled_names:
        issues.append({
            'type': 'untitled',
            'severity': 'medium',
            'auto_fix': True,
            'fix': 'generate_title_from_content'
        })
    
    return issues
```

#### 2.2. Check Placement
```python
def check_placement(note, vault):
    """
    Проверяет конфликт между:
    - Содержимым заметки
    - Текущей папкой
    - Тегами
    """
    issues = []
    
    # Извлечь основную тему из содержимого
    inferred_category = infer_category_from_content(note.content)
    
    # Извлечь категорию из папки
    folder_category = extract_category_from_path(note.path)
    
    # Извлечь категорию из тегов
    tag_categories = [extract_category_from_tag(tag) for tag in note.tags]
    
    # Сравнение
    if inferred_category != folder_category:
        issues.append({
            'type': 'folder_mismatch',
            'severity': 'high',
            'current_folder': note.folder,
            'suggested_folder': map_category_to_folder(inferred_category),
            'reason': f'Content suggests {inferred_category}, but in {folder_category}'
        })
    
    if inferred_category not in tag_categories:
        issues.append({
            'type': 'missing_category_tag',
            'severity': 'medium',
            'suggested_tag': map_category_to_tag(inferred_category)
        })
    
    return issues
```

#### 2.3. Find Duplicates
```python
def find_duplicates(notes):
    """
    Использует Jaccard similarity на уровне предложений
    """
    duplicates = []
    
    for i, note1 in enumerate(notes):
        for note2 in notes[i+1:]:
            similarity = calculate_jaccard_similarity(note1.content, note2.content)
            
            if similarity > 0.8:  # 80% совпадение
                duplicates.append({
                    'note1': note1.path,
                    'note2': note2.path,
                    'similarity': similarity,
                    'suggestion': 'Merge notes and delete duplicates'
                })
    
    return duplicates
```

#### 2.4. Check Links
```python
def check_links(note, vault):
    """
    Находит битые ссылки вида [[Note Name]]
    """
    broken_links = []
    
    # Regex для извлечения Obsidian links
    links = re.findall(r'\[\[([^\]]+)\]\]', note.content)
    
    for link in links:
        # Проверить существование файла
        target_path = vault.resolve_link(link)
        if not target_path.exists():
            broken_links.append({
                'link': link,
                'line': find_line_number(note.content, link),
                'suggestion': 'Remove link or create note'
            })
    
    return broken_links
```

#### 2.5. Check Tags
```python
def check_tags(note, tags_hierarchy):

    Проверяет, что все теги из заметки есть в TAGS_HIERARCHICAL.md
```

## Команда 3: Search

### Синтаксис

```bash
notes-manager search "<query>"
  --max-results 10
  --include-backlinks
  --scope <folder>        # Ограничить поиск папкой
  --tags <tag1,tag2>      # Фильтр по тегам
```

### Функциональность

**Трёхуровневый поиск:**

#### Level 1: Tag-based Filtering
```python
def search_level1_tags(query, vault, tags_hierarchy):
    """
    Извлечь потенциальные теги из запроса
    """
    # Keyword extraction
    keywords = extract_keywords(query)  # ['docker', 'homelab', 'настройка']
    
    # Mapping на теги
    potential_tags = map_keywords_to_tags(keywords, tags_hierarchy)
    # -> [#it/docker, #инфраструктура/homelab, #инфраструктура/развертывание]
    
    # Получить все заметки с этими тегами
    candidates = []
    for tag in potential_tags:
        notes = vault.get_notes_by_tag(tag)
        candidates.extend(notes)
    
    # Deduplicate
    return list(set(candidates))
```

#### Level 2: Keyword Search
```python
def search_level2_keywords(query, candidates):
    """
    Полнотекстовый поиск в отфильтрованных заметках
    """
    keywords = extract_keywords(query)
    scored = []
    
    for note in candidates:
        score = 0
        content_lower = note.content.lower()
        
        for keyword in keywords:
            # Подсчёт упоминаний
            count = content_lower.count(keyword.lower())
            score += count
        
        # Нормализация по длине заметки
        normalized_score = score / (len(note.content) / 1000)
        
        if normalized_score > config.search.relevance_threshold:
            scored.append((note, normalized_score))
    
    # Сортировка по score
    return sorted(scored, key=lambda x: x[1], reverse=True)
```

#### Level 3: Backlink Expansion
```python
def search_level3_backlinks(top_results, vault, max_expand=5):
    """
    Найти заметки, ссылающиеся на топ результаты
    """
    related = []
    
    for note, score in top_results[:max_expand]:
        backlinks = vault.find_backlinks(note)
        
        for backlink in backlinks:
            related.append((backlink, score * 0.5))  # Пониженный score
    
    return related
```

**Приоритизация результатов:**
```python
def prioritize_results(results, vault):
    """
    Сортировка по критериям из config
    """
    def calculate_priority(note, base_score):
        priority_score = base_score
        
        # 1. Количество backlinks (приоритет 1)
        backlink_count = len(vault.find_backlinks(note))
        priority_score += backlink_count * 0.5
        
        # 2. Tag match (приоритет 2)
        # Уже учтено в base_score
        
        # 3. Modification date (приоритет 3)
        days_since_modified = (datetime.now() - note.modified).days
        recency_score = max(0, 1 - (days_since_modified / 365))
        priority_score += recency_score * 0.3
        
        return priority_score
    
    prioritized = [
        (note, calculate_priority(note, score))
        for note, score in results
    ]
    
    return sorted(prioritized, key=lambda x: x[1], reverse=True)
```

```python
def generate_questions(vault, profile=None, n=5):
    """
    Генерирует адаптивные вопросы на основе:
    - Контекста (недавние заметки)
    - Профиля пользователя (если существует)
    - Прошлых дневниковых записей
    """
    questions = []
    
    # Базовые вопросы (fallback)
    base_questions = [
        "Что сегодня было самым важным?",
        "Чему новому научился?",
        "Какие эмоции испытывал?",
        "Что хотел бы изменить в сегодняшнем дне?",
        "О чём думал больше всего?"
    ]
    
    # Если есть профиль — адаптивные вопросы
    if profile:
        # Из недавних заметок
        recent_notes = vault.get_recent_notes(days=7, tags=['#it/*', '#карьера/*'])
        if recent_notes:

            questions.append(f"Какой прогресс по {topics[0]}?")

        if 'recurring_themes' in profile:

            questions.append(f"Как сегодня обстояли дела с {theme}?")
    
    # Заполнить до N вопросов
    while len(questions) < n:
        questions.append(random.choice(base_questions))
        base_questions.remove(questions[-1])
    
    return questions[:n]
```

```python
def find_connections(answers, vault):
    """
    Ищет связи с прошлыми дневниковыми записями
    """
    # Извлечь keywords из ответов
    keywords = extract_keywords(' '.join(answers))
    
    # Получить все дневниковые записи
    daily_notes = vault.get_notes_by_folder('10-daily/')
    
    connections = []
    
    for note in daily_notes:
        # Пропустить сегодняшнюю дату
        if note.date == datetime.today():
            continue
        
        # Вычислить keyword overlap
        note_keywords = extract_keywords(note.content)
        overlap = set(keywords) & set(note_keywords)
        
        if len(overlap) >= 2:  # Минимум 2 общих ключевых слова
            # Проверить временные паттерны
            days_ago = (datetime.today() - note.date).days
            
            # Приоритет: неделя назад, месяц назад, год назад
            if days_ago in [7, 30, 365] or 5 <= days_ago <= 9:
                connections.append({
                    'note': note,
                    'days_ago': days_ago,
                    'common_keywords': overlap,
                    'excerpt': extract_relevant_excerpt(note, overlap)
                })
    
    return sorted(connections, key=lambda x: x['days_ago'])[:3]  # Топ-3
```

```python
def analyze_cognitive_style(notes):
    """
    Системное vs интуитивное мышление
    Аналитическое vs холистическое
    """
    indicators = {
        'systematic': ['план', 'структура', 'алгоритм', 'этап', 'последовательно'],
        'intuitive': ['чувствую', 'кажется', 'интуиция', 'догадка'],
        'analytical': ['анализ', 'разобрать', 'компоненты', 'детали'],
        'holistic': ['общая картина', 'в целом', 'взаимосвязь']
    }
    
    scores = {key: 0 for key in indicators}
    
    for note in notes:
        content_lower = note.content.lower()
        for style, keywords in indicators.items():
            for keyword in keywords:
                scores[style] += content_lower.count(keyword)
    
    # Определить доминирующий стиль
    total = sum(scores.values())
    percentages = {k: (v/total)*100 for k, v in scores.items()} if total > 0 else {}
    
    return {
        'dominant': max(scores, key=scores.get),
        'distribution': percentages
    }

def find_contradictions(notes):
    """
    Находит противоречивые утверждения в разных заметках
    """
    contradictions = []

def are_contradictory(text1, text2):
    """
    Простая эвристика для определения противоречий:
    - Одинаковая тема
    - Противоположные модальности (люблю/не люблю, хорошо/плохо)
    """
    # Извлечь темы
    topic1 = extract_main_noun(text1)
    topic2 = extract_main_noun(text2)
    
    if topic1 != topic2:
        return False
    
    # Проверить полярность sentiment
    sentiment1 = analyze_sentiment(text1)  # -1 to 1
    sentiment2 = analyze_sentiment(text2)
    
    # Противоположные мнения
    if (sentiment1 > 0.5 and sentiment2 < -0.5) or \
       (sentiment1 < -0.5 and sentiment2 > 0.5):
        return True
    
    return False
```

```python
def build_knowledge_map(vault):
    """
    Граф знаний: темы, уровень глубины, связи
    """
    all_notes = vault.get_all_notes(exclude_folders=['99_Служебная'])
    
    # Построить граф
    graph = nx.DiGraph()
    
    for note in all_notes:
        # Добавить узел
        graph.add_node(note.filename, **{
            'tags': note.tags,
            'modified': note.modified,
            'word_count': len(note.content.split())
        })
        
        # Добавить рёбра (ссылки)
        links = extract_links(note.content)
        for link in links:
            target = vault.resolve_link(link)
            if target:
                graph.add_edge(note.filename, target.filename)
    
    # Анализ графа
    knowledge_map = {
        'total_notes': len(graph.nodes),
        'total_connections': len(graph.edges),
        'hub_notes': find_hub_notes(graph),  # Узлы с наибольшим количеством связей
        'topic_clusters': detect_topic_clusters(graph, vault),
        'knowledge_depth': calculate_knowledge_depth(graph)
    }
    
    return knowledge_map

def detect_topic_clusters(graph, vault):
    """
    Кластеризация заметок по темам
    """
    from sklearn.cluster import AgglomerativeClustering
    
    # Создать матрицу признаков (теги + keywords)
    notes = list(graph.nodes)
    feature_matrix = []
    
    for note_name in notes:
        note = vault.get_note(note_name)
        features = set(note.tags) | set(extract_keywords(note.content))
        feature_matrix.append(features)
    
    # Вычислить матрицу сходства (Jaccard)
    similarity_matrix = compute_jaccard_matrix(feature_matrix)
    
    # Кластеризация
    clustering = AgglomerativeClustering(n_clusters=None, distance_threshold=0.7)
    labels = clustering.fit_predict(1 - similarity_matrix)  # Distance = 1 - similarity
    
    # Группировка по кластерам
    clusters = {}
    for note_name, label in zip(notes, labels):
        if label not in clusters:
            clusters[label] = []
        clusters[label].append(note_name)
    
    # Именование кластеров
    named_clusters = {}
    for label, note_names in clusters.items():
        # Извлечь общую тему
        all_tags = []
        for note_name in note_names:
            note = vault.get_note(note_name)
            all_tags.extend(note.tags)
        
        most_common_tag = Counter(all_tags).most_common(1)[0][0]
        cluster_name = most_common_tag.replace('#', '').replace('/', ' / ')
        
        named_clusters[cluster_name] = note_names
    
    return named_clusters
```

```python
    loaded = []
    current_tokens = 0
    max_tokens = 200000 * context_threshold
    
    for note in notes:
        note_tokens = estimate_tokens(note.content)
        
        if current_tokens + note_tokens > max_tokens or len(loaded) >= max_notes:
            print(f"⚠️ Достигнут лимит контекста ({len(loaded)} заметок, {current_tokens} tokens)")
            print(f"💡 Сокращаю scope для оставшихся {len(notes) - len(loaded)} заметок")

                summary = create_summary(remaining)

                    break
            break
        
        loaded.append(note)
        current_tokens += note_tokens
    
    return loaded
```

### A. Keyword Extraction
```python
def extract_keywords(text, top_n=10):
    """
    TF-IDF based keyword extraction
    """
    # Tokenization
    tokens = word_tokenize(text.lower())
    
    # Remove stopwords
    stopwords = load_stopwords('ru') + load_stopwords('en')
    filtered = [t for t in tokens if t not in stopwords]
    
    # Calculate TF-IDF
    tf = Counter(filtered)
    idf = calculate_idf(filtered, corpus=all_vault_notes)
    
    tfidf_scores = {word: tf[word] * idf[word] for word in tf}
    
    # Return top N
    return sorted(tfidf_scores.items(), key=lambda x: x[1], reverse=True)[:top_n]
```

### B. Jaccard Similarity
```python
def calculate_jaccard_similarity(text1, text2):
    """
    Jaccard similarity coefficient
    """
    set1 = set(word_tokenize(text1.lower()))
    set2 = set(word_tokenize(text2.lower()))
    
    intersection = len(set1 & set2)
    union = len(set1 | set2)
    
    return intersection / union if union > 0 else 0
```

### C. Graph Traversal для поиска связей
```python
def find_related_notes(start_note, vault, max_depth=3):
    """
    BFS для поиска связанных заметок
    """
    visited = set()
    queue = [(start_note, 0)]  # (note, depth)
    related = []
    
    while queue:
        current, depth = queue.pop(0)
        
        if current in visited or depth > max_depth:
            continue
        
        visited.add(current)
        related.append((current, depth))
        
        # Get linked notes
        links = vault.get_links(current)
        backlinks = vault.get_backlinks(current)
        
        for link in links + backlinks:
            if link not in visited:
                queue.append((link, depth + 1))
    
    return related
```
